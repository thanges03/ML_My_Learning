# Linear Regression

## Project Overview
This project demonstrates the implementation of **Linear Regression** using a synthetically generated (random) dataset.

---

## Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

## Dataset Description
- Dataset is **synthetically generated**
- Formula used:

y = 3x + 5 + noise

---

## Model Workflow
1. Generate random data  
2. Split data into training and testing sets  
3. Train Linear Regression model  
4. Predict on test data  
5. Evaluate using:
   - Mean Squared Error (MSE)
   - RÂ² Score  
6. Visualize regression line
